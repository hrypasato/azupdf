# azupdf

Aplicación web para extraer y visualizar documentos PDF desde visores embebidos (iframe) en sistemas que no permiten descarga directa.

🔗 Demo en producción:  
https://azupdf.streamlit.app/

---

## 🚀 Características

- 🔍 Extrae automáticamente el PDF desde páginas con visor
- 📂 Abre el documento en una nueva pestaña
- 👁️ Visualiza el PDF directamente en la app
- ⚡ Interfaz simple (solo pegar enlace)
- 🧠 Funciona sin necesidad de conocimientos técnicos

---

## 🛠️ Tecnologías utilizadas

- Python
- Streamlit
- Requests
- BeautifulSoup
